📊 Manufacturing Analytics Dashboard
📌 Overview

The Manufacturing Analytics Dashboard is designed to provide actionable insights into production performance, quality metrics, and operational efficiency. It enables stakeholders to monitor key KPIs, identify bottlenecks, and make data-driven decisions.


🎯 Objectives
Track production output and efficiency
Monitor rejection rates and quality issues
Analyze department-wise and employee performance
Identify operational bottlenecks
Enable real-time decision-making



📈 Key Features
Production Metrics: Manufactured, processed, and produced quantities
Quality Analysis: Rejection trends and repeat defects
Performance Tracking: Employee and machine efficiency
Department Insights: Department-wise comparison
Time-based Analysis: Daily and period-wise trends
Interactive Filters: Drill-down across multiple dimensions
